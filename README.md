# Bohdan Zelenskyi Portfolio

Freelance portfolio website for Bohdan Zelenskyi, built with Next.js.

The site is designed for clients and presents:

- Home page with strong positioning and CTA
- About page
- Work experience page
- Projects page
- Tech stack page
- Contact page
- Resume download

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production checks

```bash
npm run lint
npm run build
```
